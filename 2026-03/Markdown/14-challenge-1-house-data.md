## Premise of the Challenge 

**Mozilla's: Building a house data UI**
- Used JavaScript to implement search and filtering for a property website UI.
    - involves fetching JSON data
    - filtering the data based on values entered 
    - rendering the data to the UI 


## What concepts did it reinforce:

Through this exercise I used several different concepts such as: 

1. Async Functions
    - fetch() / await 
2. Loops
    - for / forEach
3. Conditional statements (if statements)
    - different operators: || and &&
4. Array methods
    - filter()
    - includes()
5. Try / Catch block for error handling 



## Were there any challenges I faced? (March 15, 2026)

There were a couple times I was tripped up by different syntax (how things should be written). I would understand the concept, but I wouldn't fully understand how to implement it. An example of this would be for looping through objects, and how the syntax is slightly different from looping through an array. Or when I was writing my conditional statements using '||' and '&&'. As I haven't used them that often, I didn't realise that the positions you put them in can affect how the statement is evaluated and that each condition should be put in parenthesis.
