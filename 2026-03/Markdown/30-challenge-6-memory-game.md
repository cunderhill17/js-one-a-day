## Premise of the Challenge 

1. I'm aiming to make a mini game where the user is shown a sequence of colors for a short period of time and then has to repeat the sequence. For every new sequence, the amount of time the colors are shown for gets shorter. 
2. I'm going to keep it simple, and do 6 colours, and each sequence will be 5 colours long



## What concepts did it reinforce:

1. I was able to use `setTimeout` for the first time without having to look it up, though I did have to loop up how to clear previous uses
2. for loop 
    - used to add colors to an array 
    - also used to compared the user and system arrays 
3. Event listeners 
4. Conditional statements (if)
5. Creating HTML elements with JS and adding them to the DOM


## Were there any challenges I faced? (March 31, 2026)
